# ✈️ Sistema de Gerenciamento de Tráfego Aéreo

## 📌 Descrição do Projeto

Este projeto está sendo desenvolvido como parte das atividades acadêmicas da **Faculdade Católica Imaculada Conceição do Recife (FICR)**, pelos alunos do programa **Embarque Digital**.

Este projeto tem como objetivo o desenvolvimento de um sistema de gerenciamento de tráfego aéreo, capaz de simular e/ou controlar o fluxo de aeronaves em um determinado espaço aéreo.

A proposta é explorar conceitos como:

* Controle de rotas aéreas
* Gerenciamento de pousos e decolagens
* Prevenção de colisões
* Organização do espaço aéreo

Este projeto está em fase inicial de desenvolvimento e ainda não possui implementação completa.
